# mvim

## Requirement
OS: Ubuntu

git

## installation
```bash
git clone https://github.com/TakutoYoshikai/mvim.git ~/.mvim
echo "export PATH=\$PATH:$HOME/.mvim/bin" >> ~/.bash_profile
source ~/.bash_profile
```

## how to use
open file
```bash
tvim <file>
```

show history and open file
```
tvim -l
#enter file number
```

reset history
```bash
tvim -r
```
